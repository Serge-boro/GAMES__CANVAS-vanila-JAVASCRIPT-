In this JavaScript example game I aspired to use a lot of different sprite sheet to animate randomised characters created with JavaScript ES6 class.
Image comes from general Google search, used for educational purposes only.
INSTRUCTION GAME:
You can use your computer mouse for making choices in local menu options.
Move heroe:
"a" - turn left,
"d" - turn right,
"space" - jump,
"r" - magic ball,
"esc" - pause
